# Коронавирус, уходи!

Bootstrap guide:</br>

- `pip install pygame`.</br>

Server:</br>
1. `python3 server.py`
2. tell clients your `public IP`.

Client:</br>
1. figure out server `public IP`.
3. `python3 client.py`.
3. enter `public IP` you received on step 1.
4. **enjoy!**

Tip: to kill stopped server process on Linux & Mac use `kill -s SIGKILL $(jobs -p)`
